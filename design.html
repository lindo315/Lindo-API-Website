STYLE GUIDE

This is my Style Guide which features more my design choices.

Color Scheme

  • Background: The website's background features a photo in the landing page. It's a photo of an active Solar Flare on the sun, a way to introduce what the website is about. Further down the website, the background color is a dim white-ish color which is easier on the eyes but still makes the content of the website pop out.
  • Accents: To add visual interest to the website, I added bold accent colors throughout, including orange text on white, some darker colors for the nav-bar and orange images to represent the sun to an extent.
  • HEX Values:
    • Background: #fafafa
    • Accents: #ff6600, #000C40
  • Usage Guidelines:
    • Button: #d1d7db, #c5cace
    • Links (Tabs): #ff6600
    • Headings: #ff6600,#000C40

Typography

  • Headings and subheadings: For the headings I used a NASA font with a Helvetica to complement it: 'Nasa', sans-serif & 'Helvetica Neue', sans-serif. I chose this fonts because the website is inspired by NASA and they use the same fonts.
  • Body text: For the body text on my website, I've chosen to use Arial. This font is a classic choice for body text and is easy to read, making it a great choice for my website content.
  • Font Sizes:
    • Headings: 80px (H1), 60px (H2), 30px (H3)
    • Body Text: 30px
  • Line Heights:
    • Headings: 1.2em
    • Body Text: 1.5em
    • Subheadings: 1.3em

Imagary

  • Photos: In this website I have two images on the on the main page and a few wireframes on the design section. For the landing page, I chose a solar flare image as mentioned before, and for the solar flares section of the website I chose one image that illustrates the formation of solar flares.
  • Illustrations: There are two line Illustrations on the about page for aesthetics and several other images/illustrations of solar flares developing on the solar flares section, these were useful is visually portraying the formation and developing of solar flares.
  • Image Sizes: I've optimized image sizes to fit their required space. The homepage background image fits the entire screen, while the solar flare illustrations take up a small space right next to the text they represent.
  • Image Formats: I chose to use a JPG and PNG format for the images.

Layout

  • Grid System: I created a grid system for the formation of solar flares illustration with the texts and icons, I wanted to style it in an aesthetically pleasing was so I made it into an alternating grid.
  • White Space: In this website I used a lot of white space, it was part of my design choice, or aesthetic. The website is not too fancy or distractingly styled, so the white space works with the minimalist design.
  • Navigation: To improve user experience, I implemented a clear and easy-to-use navigation system. The navigation is located at the top of the page, it also glows or changes color when the user exits or enters a different section to show them where they are in the page.

My Goal for this Website

My goal was to create a simplistic website that doesn't focus too much on style but make sure to coomunicate what it's about. Since I chose to work with a solar flare API I wanted to put some colors like orange or yellow, maybe black for space, for aesthetic purposes. But that was going to cost me time so I decided to focus my time on implimenting the API, without losing sight on making the website pleasing to look at.
